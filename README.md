## Usage

Just type any request starting with `how`:

```bash
how to create a git branch
```

```bash
how to convert video to gif with ffmpeg
```

```bash
how to upgrade node to latest version
```

<img src="./demo.gif">

## Development

To install dependencies:

```bash
bun install
```

To run:

```bash
bun how
```

To compile:

```bash
bun compile
```
